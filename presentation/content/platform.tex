\begin{frame}
\frametitle{Arkitektur - Kriterier}
Kriterierne dækkes af følgende elementer i arkitekturen:
\begin{itemize}
\item Modulær -> Moduler
\item Fleksibel -> Manager
\item Kombiner -> Moduler
\item Kommunikativ -> DB Access/DB
\end{itemize}
\end{frame}
